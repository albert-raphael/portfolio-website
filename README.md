# Personal Portfolio Website

## Project Overview
A clean, responsive portfolio website built with pure HTML and CSS to showcase data analysis projects and professional information.

## Live Demo
[View Live Website](https://your-username.github.io/portfolio-website)

## Project Description
This is a single-page portfolio website designed by Albert Raphael Assimagbe, a Data Analyst and Economist. The website showcases professional skills, project portfolio, and contact information in a modern, mobile-friendly design.

## Features
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Clean Layout**: Professional appearance with clear visual hierarchy
- **Project Showcase**: Display of 4 data analysis projects with images and descriptions
- **Contact Integration**: Direct email functionality and social media links
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Accessibility**: Keyboard navigation and screen reader friendly

## Technologies Used
- **HTML5**: Semantic markup and modern structure
- **CSS3**: Flexbox, Grid, responsive design, and animations

## How to Run the Project

### Local Development
1. Download all project files
2. Ensure images are in the `images` folder
3. Open `index.html` in any web browser
4. No server or build process required

### File Requirements
- All images must be in the `images` folder
- CSS file must be named `style.css`
- HTML file must be named `index.html`

## Customization Guide

### Changing Personal Information
1. Update name and title in hero section
2. Modify about me text in about section
3. Replace profile image in images folder

### Adding New Projects
1. Add new project card in projects section
2. Include project image in images folder
3. Update project description and links

### Modifying Colors
1. Update CSS color variables in `:root`
2. Change button and background colors
3. Maintain contrast for accessibility
